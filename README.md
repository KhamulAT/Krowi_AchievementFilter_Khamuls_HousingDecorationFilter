This addon provides additional filters for Krowi's AchievementFilter Addon.

Includes all known decors, gathered through Achievements, added with 11.2.5

- Dragonflight
- Shadowlands
- Battle for Azeroth
- Legion
- Warlords of Draenor
- Mists of Pandaria
- Cataclysm
- Wrath of the Lichking
- Burning Crusade?
- Vanilla
- Class Specific
- PvP Achievements
